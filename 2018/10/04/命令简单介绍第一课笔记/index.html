命令简单介绍第一课笔记

第一：
相对路径和绝对路径
相对路径:从当前路径开始进入blog文件夹

cd blog

绝对路径：从跟目录开始进入blog文件夹

cd /home/yq/Desktop/blog  全称：根目录下home文件下的yq文件下的Desktop文件下的blog文件夹

    cd ./blog：进入当前路径下的blog文件夹

cd .   当前路径
cd .   .当前路径的上层路径
cd ../.. 进入上两层路径
cd -   快速回到上次目录所在的路径
cd ~   返回home路径

第二：

pwd: 显示完整路径
clear：清理屏幕
cat:查看文件内容

第三：重定向
cat 1.txt > 3.txt 将1.txt的内容写到3.txt中，同时删除原来3.txt中的内容

cat 1.txt >> 3.txt 将1.txt的内容写到3.txt中，不删除原来3.txt中的内容

cat 合并文件

cat 1.txt 2.txt >> 3.txt
将1.txt和2.txt中的内容合并到3.txt中。

第四： more 命令
more xx.txt
将txt文件一部分一部分显示出来
eg: more 2.txt
查看一个目录多文件
eg: 将bin中文件罗列到 5.txtz中 ls /bin >>5.txt
或者也可以 通过ls /bin | more来分步显示
将原来显示的内容显示到|中，可以理解|为显示的水管

第五：mkdir/rmdir 创建/删除目录
mkdir blog 创建blog文件夹

如果创建的文件的依赖文件夹不存在可以在命令后面加-p这样就会自动创建这些文件夹
eg: mkdir /A/B/C/D -p

rmdir

删除文件夹

rmdir -C -r 即使C中有文件那也能强制删除

第六：tree 看路径下的文件情况

第七：软/硬链接类似于win快捷建

软链接
ln -s 源文件 对应的软连接（win快捷建）

ln -s 1.txt 1-softlink.txt

硬链接

ln 1-1.txt 1-1-hardink.txt

软硬链接区别
软连接源文件删除后，链软接就没有数据啦；硬链接则不是。
